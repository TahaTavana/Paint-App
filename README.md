# 🎨 Paint App (Python)

A simple desktop paint application built using Python.

## 🖌️ Features

- Freehand drawing with the mouse
- Color selection
- Adjustable brush size
- Eraser tool
- Clear the canvas
- Easy-to-use graphical user interface (GUI)
